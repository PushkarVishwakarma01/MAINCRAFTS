# RPM // Full-Stack Engine 🚀

A futuristic full-stack landing page built using **HTML, CSS, Java Spring Boot**, and a **Contact Form** that sends data from the frontend to the backend.

This project demonstrates the complete workflow of:

Frontend Form → POST Request → Spring Boot Backend → Console Output

---

# 📌 Features

✅ Modern futuristic landing page UI  
✅ Responsive contact form  
✅ Spring Boot backend integration  
✅ Handles POST requests  
✅ Sends user data from frontend to backend  
✅ Prints submitted data in console  
✅ Beginner-friendly full-stack project  

---

# ⚙️ How It Works

1. User fills the contact form.
2. Form sends a POST request to:
   /contact
3. Spring Boot controller receives the data.
4. Backend prints submitted information in console.
